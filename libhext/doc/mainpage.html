Rough overview

hext::Parser

hext::Parser lexes strings containing hext rule definitions with a state-machine generated by Ragel (see hext-machine.rl). hext::Parser is a simple combination of a lexer and the builder pattern: The lexer uses hext::RuleBuilder to construct hext::Rule. This approach is probably sufficient because hext rule definitions are very simplistic.

hext::Rule

A hext::Rule defines how to match and capture html-nodes. It can be applied to a GumboNode tree, where it recursively tries to find matches. Once a match is found, data is extracted.

hext::Html

hext::Html is a RAII wrapper for Gumbo Parser, an HTML5 parsing library written in C99.
